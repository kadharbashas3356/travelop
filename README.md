This project analyzes airline passenger data to evaluate customer satisfaction and service performance. The analysis focuses on identifying key factors that influence passenger experience, including flight delays, service quality, booking process, and onboard facilities.

Using SQL for data extraction and transformation and Microsoft Power BI for visualization, the project builds interactive dashboards to track key performance indicators (KPIs) such as satisfaction levels, delay impact, and service ratings.

The insights generated from this analysis help identify areas for operational improvement and support data-driven decision-making to enhance customer experience in the airline industry.
